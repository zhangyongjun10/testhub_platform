## 1. 项目概述

### 1.1 项目简介

TestHub 是一个 AI 驱动的企业级测试管理平台，提供从需求分析到测试执行、报告生成的完整测试生命周期管理解决方案。

### 1.2 核心功能

- **测试用例管理**：支持功能、API、UI、性能、安全等多种测试类型
- **API 测试**：HTTP/WebSocket 接口测试，支持自动化和定时任务
- **UI 自动化**：基于 Selenium/Playwright 的 Web UI 自动化测试
- **AI 需求分析**：智能解析需求文档，自动生成测试用例
- **测试执行**：手动/自动化测试执行，结果跟踪和分析
- **报告管理**：集成 Allure 生成专业测试报告
- **智能助手**：基于 Dify AI 的测试助手

### 1.3 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | Django 4.2, Django REST Framework |
| 前端框架 | Vue 3.3, Vite 7 |
| 数据库 | MySQL 8.0+, Redis |
| 认证 | JWT (SimpleJWT) |
| 异步任务 | Celery, Django Channels |
| AI集成 | browser-use, LangChain, httpx |
| 自动化 | Selenium, Playwright, AirTest |

### 9.3 部署方式

#### 9.3.1 Docker部署 (推荐)

```yaml
# docker-compose.yml
version: '3.8'
services:
  db:
    image: mysql:8.0
    environment:
      MYSQL_DATABASE: testhub
      MYSQL_ROOT_PASSWORD: ${DB_PASSWORD}
    volumes:
      - mysql_data:/var/lib/mysql

  redis:
    image: redis:6-alpine

  backend:
    build: ./backend
    command: gunicorn config.wsgi:application
    depends_on:
      - db
      - redis

  celery:
    build: ./backend
    command: celery -A config worker
    depends_on:
      - db
      - redis

  frontend:
    build: ./frontend
    ports:
      - "80:80"
```

# 后端部署

gunicorn config.wsgi:application -b 0.0.0.0:8000

# 前端部署
